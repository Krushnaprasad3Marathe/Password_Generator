# Password Generator (Python)

This is a simple Password Generator program written in Python.
It generates strong passwords using numbers, letters, and symbols.

## Features
- Generate password of length 6, 7, or 8
- Uses:
  - Numbers
  - Uppercase letters
  - Lowercase letters
  - Special characters
- Option to store password for a website
- Beginner friendly project

## Technologies Used
- Python 3
- random module

## How to Run

1. Clone the repository:
   git clone https://github.com/Krushnaprasad3Marathe/Password_Generator.git

2. Go to project folder:
   cd Password_Generator

3. Run the program:
   python password_generator.py
